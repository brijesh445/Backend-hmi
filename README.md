# Backend - HMI

Backend-HMI is a RESTful API design using Node.js, Express.js, Typescript, and MongoDB.

## Installation

Use the package manager Yarn or Npm to install libraries

```bash
yarn 
```


```bash
npm install
```

## env varibale

```bash
PORT=3000
ATLAS_URI=MONGO_URI
```

## Run

```bash
yarn run dev 
```
